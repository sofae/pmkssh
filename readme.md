PMKSSH

windows桌面软件  
以轻量化为初衷的远程管理工具  
用于批量远程管理服务器  
基于ansible内核paramiko使用python开发  
  
1.通过输入服务器信息加密保存到数据文件  
2.数据文件绑定电脑安全无泄漏  
3.单选或多选服务器后输入命令点击运行  
4.如填写单行命令,则命令文件不执行  
5.导入服务器信息文件格式ip,port,user,pwd以TAB分隔  

程序打包  
pyinstaller -F pmkssh.py
